📦 Project Overview

This project demonstrates a complete CI/CD pipeline that automatically builds, tests, containerizes, and deploys a web application to Kubernetes.

🛠 Tools & Technologies Used

GitHub | Jenkins | SonarQube | Docker | AWS ECR | Kubernetes | Argo CD

⚙️ CI/CD Workflow

1. Developer pushes code to GitHub
2. Jenkins triggers the CI pipeline
3. SonarQube performs code quality analysis
4. Docker image is built and pushed to AWS ECR
5. Kubernetes deployment manifest updated
6. Argo CD automatically deploys to the cluster

☁️ Deployment Environment

Platform: Kubernetes Cluster (EKS / K8s)
Container Registry: AWS ECR
Deployment Strategy: GitOps using Argo CD
